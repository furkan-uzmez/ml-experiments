# Phase 3 Plan: U-Net Baseline Pipeline

## Goal
Implement the U-Net benchmark pipeline as the simplest supervised baseline, validating the full train‚Üíinfer‚Üíeval flow and ensuring robust baseline performance on ISIC 2018.

## Requirements Covered
UNET-01, UNET-02, UNET-03, UNET-04, UNET-05, REPRO-01, REPRO-03

## Approach (select-skills.md & python-expert)
Following `python-expert` guidelines, all code will be strictly type-hinted, PEP-8 compliant, and modular. We will use `MONAI` for the U-Net architecture as it is the industry standard for medical imaging and handles variable shapes elegantly.

## Tasks

### 1. Configuration: `configs/unet.yaml`
Define hyperparameters:
- `model`: MONAI BasicUNet (2D, in_channels=3, out_channels=1)
- `batch_size`: 16
- `epochs`: 50
- `optimizer`: AdamW (lr=1e-4)
- `loss`: DiceFocalLoss
- `resize`: 256x256
- `augmentation`: RandRotate, RandFlip, RandZoom

### 2. Model & Data: `src/models/unet_model.py` & `src/dataio/unet_dataset.py`
- `src/models/unet_model.py`: Factory function returning a MONAI `BasicUNet`.
- `src/dataio/unet_dataset.py`: PyTorch `Dataset` wrapping `dataset_index.py` and `split_loader.py`. Uses `monai.transforms` for reading, resizing (256x256), and normalization.

### 3. Training Loop: `src/train/train_unet.py`
- Implements standard PyTorch training loop.
- Uses `configs/seeds.yaml` to enforce deterministic seeds.
- Runs 3 separate training sessions for seeds `11`, `22`, `33`.
- Saves best checkpoints based on validation Dice to `artifacts/unet/checkpoints/seed_X/`.

### 4. Inference & Output: `src/infer/infer_unet.py`
- Loads best checkpoint for a given seed.
- Decorates with `@track_inference_time` and `@track_peak_gpu_memory`.
- Inherits evaluation capabilities from `src/evaluation/resample.py` to restore predictions to native 2D resolution.
- Saves raw binary masks as PNG to `artifacts/unet/predictions/seed_X/<patient_id>.png`.

### 5. Automation: `scripts/run_unet_benchmark.sh`
- A master script tying it all together:
  1. Train Seed 11, 22, 33
  2. Infer Seed 11, 22, 33 (generating predictions)
  3. Run evaluation (Dice/IoU/HD95) for each seed (will use Phase 2 scaffold)

## Verification
- Write `tests/test_unet_model.py` to ensure shape safety (B, 3, 256, 256 -> B, 1, 256, 256).
- Write `tests/test_unet_dataset.py` to verify transforms don't destroy masks.
- Manually run `scripts/run_unet_benchmark.sh` for 1 epoch to ensure end-to-end functionality without crashing.
