# Phase 6 Plan: Results Aggregation & Report

## Goal
Automatically aggregate all evaluation metrics JSON files generated from the pipeline runs (U-Net, nnU-Net, MedSAM), calculate statistical confidence intervals (where applicable based on multiple seeds), and automatically generate standardized visualization plots and a final Markdown report.

## Requirements Covered
REPORT-01, REPORT-02, REPORT-03, REPORT-04, REPORT-05

## Architecture & Integration

```
src/
└── reporting/
    ├── aggregate_metrics.py      # Parses all prediction JSONs into a unified Pandas DataFrame
    ├── visualize.py              # Generates Boxplots, Bar Charts (Matplotlib & Seaborn)
    └── generate_report.py        # Compiles benchmark_report.md
reports/
├── figures/                      # Auto-generated .png charts
└── BENCHMARK_REPORT.md           # The final evaluation artifact
scripts/
└── run_final_reporting.sh        # Bash orchestrator
```

### 2. Visualization (`src/reporting/visualize.py`)
- **Metric Plots**: Boxplots for Dice, IoU, and HD95 comparing U-Net, nnU-Net, and MedSAM side-by-side. (Averaged across seeds for U-Net/nnU-Net).

### 3. Report Generation (`src/reporting/generate_report.py`)
- Write a Markdown jinja-like template string.
- Automatically insert aggregate statistics (Mean ± Std) into a Markdown table.
- Link the generated `reports/figures/` images into the Markdown document.
- Briefly auto-summarize the "Winner" for Dice metric.

### 4. Pipeline Finalization (`scripts/run_final_reporting.sh`)
- A single bash script that triggers the aggregation, visualization, and markdown generation steps.
